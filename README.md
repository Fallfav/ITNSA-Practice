# 🏆 ITNSA-Practice

A structured repository for practicing and documenting skills related to **IT Network Systems Administration (ITNSA)** — one of the competency areas in LKS (Lomba Kompetensi Siswa) / WorldSkills Indonesia.

---

## 📁 Repository Structure

This repository is organized into focused branches to keep each practice area clean and modular.

| Branch | Description |
|---|---|
| `main` | Overview, documentation, and general resources |
| `Cisco-Practice` | Cisco networking configurations (routing, switching, ACL, etc.) |
| `Server-Environment-Practice` | Server setup and environment configuration (Linux, Windows Server, services) |

---

## 🌿 Branches

### `Cisco-Practice`
Covers hands-on practice with Cisco devices and simulations, including:
- Routing protocols (OSPF, EIGRP, BGP)
- VLANs, STP, and switching concepts
- Access Control Lists (ACL)
- Network Address Translation (NAT)
- Cisco IOS configuration & troubleshooting

### `Server-Environment-Practice`
Covers server configuration and environment setup, including:
- Linux server administration (Ubuntu/Debian/CentOS)
- Windows Server roles and features
- Web, DNS, DHCP, FTP, and mail server setup
- Firewall and security hardening
- Virtualization and containerization basics

---

## 📌 How to Navigate

1. Clone the repository
   ```bash
   git clone https://github.com/<your-username>/ITNSA-Practice.git
   ```

2. Switch to a specific branch
   ```bash
   git checkout Cisco-Practice
   # or
   git checkout Server-Environment-Practice
   ```

3. Explore the folder structure and follow the documentation inside each branch.
